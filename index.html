Cloudflare Worker — Proxy CORS

O GeoSampa bloqueia chamadas diretas de browsers (CORS).
Os proxies públicos (allorigins, corsproxy) bloquearam permanentemente o domínio prefeitura.sp.gov.br.

O único proxy confiável é o seu Cloudflare Worker próprio.

URL do seu Worker
Worker parou de funcionar?
Causas comuns:
· Cloudflare pausou o Worker por inatividade
· Limite de 100k req/dia do plano free atingido
· A URL do Worker mudou

Solução rápida:
1. Acesse dash.cloudflare.com
2. Workers & Pages → seu worker
3. Verifique se está Active (não Disabled)
4. Clique em "Manage" → confirme o deploy
5. Volte aqui e clique "Testar conexão"
Criar novo Worker (5 min)
1. cloudflare.com → conta grátis
2. Workers & Pages → Create → Hello World
3. Deploy → Edit code → cole o worker.js → Deploy
4. Copie a URL gerada e cole acima
5. Clique "Testar conexão" — deve ficar verde
